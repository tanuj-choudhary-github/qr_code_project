# qr_code_project

## 📖 Project Description
This is a simple **QR Code Generator project built using Python**.  
The program creates a QR code image that can be easily scanned using any QR scanner to reveal the embedded text.

The project also allows customization of:
- 🎨 Color (QR & background)
- 📏 Size (box size)
- 🔲 Dimensions (version)
- 📦 Border thickness

---

## 🚀 Features
- Generate QR codes instantly  
- Customizable design (color & size)  
- High error correction support  
- Easy to use and beginner-friendly  

---

## 🛠️ Technologies Used
- Python  
- qrcode library  
- Pillow (PIL)  



## 🖼️ Output
The program generates a QR code image file:

`tanuj_choudhary.png`  
Scan it using any QR scanner to view the text.
